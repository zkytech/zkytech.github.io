---
title: 常用快捷键
---

| 快捷键          | 说明                                                         |
| --------------- | ------------------------------------------------------------ |
| `F9`            | 更新`域`。所有自动编号都是`域`，对自动编号项目的交叉引用也是`域`，目录也是`域`。这些对象都可以通过`F9`来更新 |
| `Alt+=`         | 插入公式                                                     |
| `Ctrl+Shift+= ` | 上标                                                         |
| `Ctrl+=`        | 下标                                                         |
| `Ctrl+E`        | 居中对齐                                                     |

`F9`常常搭配`Ctrl+A`使用，可以快速更新全文文本中的域，但**[无法更新图片题注中的编号(域)](/论文排版指南/踩坑.html)。**

word中任何按钮都可以在其上面`右键 -> 添加到快速访问工具栏`，之后Word顶部就会出现该功能的快捷按钮。
